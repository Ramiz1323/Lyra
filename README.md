# 🌌 Lyra

> **The shortest path from curiosity to clarity.** > Lyra is a high-velocity AI discovery engine that synthesizes the open web into precise, cited insights.

---

## ✨ Overview
Traditional search gives you links; **Lyra** gives you answers. By leveraging advanced LLM synthesis and real-time web indexing, Lyra cuts through the noise to provide structured truth.

### 🚀 Key Features
* **Real-time Synthesis:** Answers generated from the latest web data.
* **Source Transparency:** Every claim is backed by clickable citations.
* **Minimalist UI:** Designed for focus and speed.
* **Contextual Memory:** Follow-up questions that understand the conversation flow.
